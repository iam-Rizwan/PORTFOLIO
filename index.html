DESIGNCAFE_CRM_REDESIGN

DesignCafe CRM Redesign

A branded CRM for an interiors company, rebuilt from a single tangled file into a clean, modular product.

Case Study

01
Problem

DesignCafe's CRM felt generic — nothing like the interiors brand it represented.

02
Research

Traced serviceability logic across Hyderabad and Mumbai pincodes, and audited the brand's visual language.

03
Ideation

Designed a three-tier lookup: hardcoded metros, an India Post API fallback, Haversine distance as backup.

04
Wireframes

Anchored the flow around a Three.js isometric room as the login's hero moment.

05
UI Design

Built a frosted-glass interface matched panel-for-panel to DesignCafe's brand references.

06
Testing

Stress-tested pincode edge cases across both metros and confirmed the 3D scene held up on device.

07
Outcome

Shipped as a clean, modular CRM — split into index.html, styles.css, and script.js with zero logic lost.
